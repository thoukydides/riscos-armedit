*ARMEdit_Clients
*ARMEdit_Clients
List currently registered clients

Syntax:*ARMEdit_Clients

Parameters:None

Use: This command displays details of the clients that have registered with the ARMEdit module for communication purposes.

Example:*ARMEdit_Clients

Related commands:None
