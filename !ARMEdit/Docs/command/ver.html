ARMEdit: *Commands

*ARMEdit_Version
*ARMEdit_Version
Display PC card version details

Syntax:*ARMEdit_Version

Parameters:None

Use: This command displays the version numbers of the data structures used by the most recently used PC front-end. It also displays the range of versions catered for by this version of the module.

Example:*ARMEdit_Version

Related commands:None
