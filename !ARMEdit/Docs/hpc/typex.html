HPC_ARMEDIT_TYPEEXT
(HPC &0105:&0006)
Convert a RISC OS filetype into a DOS extension

Data sent:0HPC service ID = &0105
+2reason code = &0006
+4

Data returned:0return code
+4

Use: Convert a RISC OS filetype into a DOS extension.

Related services:
