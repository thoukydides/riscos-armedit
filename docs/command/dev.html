ARMEdit: *Commands

*ARMEdit_Devices
*ARMEdit_Devices
List device driver details

Syntax:*ARMEdit_Devices

Parameters:None

Use: List the RISC OS path of all active emulated devices. Several other details are also listed for each device.

Example:*ARMEdit_Devices

Related commands:None
