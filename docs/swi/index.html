ARMEdit: SWI Calls

SWI Calls

The following SWIs are implemented by the ARMEdit module. For more details regarding the use of the communciations SWIs see the Code documentation.
  • ARMEdit_ControlPC
  • ARMEdit_TalkStart
  • ARMEdit_TalkEnd
  • ARMEdit_TalkTX
  • ARMEdit_TalkRX
  • ARMEdit_TalkAck
  • ARMEdit_HPC
  • ARMEdit_Polling
  • ARMEdit_TalkReply

A comprehensive OSLib interface to all of the ARMEdit SWIs and WIMP messages is included as part of the source code release.
